# Framework Público de Resiliencia Operativa - Casadiego & Asociates

Bienvenido al repositorio de recursos abiertos de **Casadiego & Asociates**. Este espacio ha sido diseñado para proveer a los equipos de tecnología, operaciones y directores de planta en LATAM herramientas base para evaluar la continuidad de sus operaciones bajo estándares internacionales (**ISO 22301** e **ITIL v4**).

## 🛡️ Propósito
Ayudar a las organizaciones a identificar puntos únicos de fallo (SPOF), mitigar el impacto financiero de la inactividad por cortes de energía (Blackouts) y blindar sus activos de información frente a amenazas lógicas como el Ransomware.

## 📁 Recursos Disponibles
* **`auditoria-resiliencia-base.md`:** Checklist maestro para auditoría rápida de infraestructura y sistemas críticos.
